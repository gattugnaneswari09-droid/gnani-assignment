# 🤖 AI-Powered Automated Assignment Evaluation System

A sophisticated assignment evaluation system powered by **Google Gemini AI** that automatically evaluates student assignments with detailed feedback, scoring, and analysis.

## 🚀 Features

### AI-Powered Evaluation
- **Intelligent Analysis**: Uses Google Gemini AI for deep understanding
- **Multi-dimensional Scoring**: Evaluates accuracy, completeness, clarity
- **Detailed Feedback**: Provides actionable insights for improvement
- **Strengths & Weaknesses**: Identifies what students did well and areas to improve

### File Support
- **Multiple Formats**: TXT, PDF, DOCX file uploads
- **Text Input**: Direct text entry for questions and answers
- **Automatic Extraction**: Extracts text from uploaded files

### Question Types Supported
- 📝 Descriptive / Essay
- ✅ Multiple Choice
- ✏️ Short Answer
- 💻 Coding / Programming
- 🧮 Problem Solving

### Database Features
- **MongoDB Storage**: All evaluations stored permanently
- **History Tracking**: View all past evaluations
- **Statistics Dashboard**: Track performance metrics
- **Question Bank**: Save and reuse questions

## 📋 Prerequisites

- Python 3.8+
- MongoDB (running on localhost:27017)
- Google Gemini API Key (included in code)
